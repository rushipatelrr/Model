# Model
This project analyzes the relationship between Bitcoin market sentiment (Fear/Greed) and trader behavior & performance on a crypto derivatives platform.
The objective is to understand:
1. Does trsder profitability change during fear vs greed days ?
2. DO traders alter behaviour?
3. Can sentiment + behaviour features predict trader profitablty ?

This project includes:
1.Data cleaning and alignment
2.Behavioral feature engineering
3.Performance comparison
4.Predictive modeling
